# Bootstrap-Color-Extension
***Css de elementos Basicos de Boostrap 4/5 con nuevos colores***

## Agrega los colores a los elementos:
- Botones(btn)
- Botones Outline(btn-outline)
- Bordes(border)
- Texto(text)
- Fondo(bg)
- Alertas(alert)
- Lineas(hr)
